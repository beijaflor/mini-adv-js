# mini-adv-js

## 目的

ステップを見ながらjavascriptでアドベンチャーゲームの作り方が学習できるようなお勉強用レポジトリ。

個人的にはgithubの使い方に慣れるのも目的。

## ステップ

  1. 画像を入れ替える

  2. テキストを入れ替える

  3. 1.と2.をセットで切り替える

  4. 配列にシナリオを用意する

  5. シナリオを順番に表示する

  6. シナリオ終了時に別URLに遷移する

  7. シナリオで次のシーンを指定できるようにする

  8. 次のシーンを複数指定できるようにする

  9. 次のシーンが2つだった場合、選択肢を表示する

  10. 選択肢に応じて次のシーンを変更する

  11. 選択肢の表示を整える

  12. テキストの表示方法を調節する

  13. 画像の指定した位置をクリックできるようにする
